# Phishing Detection and Awareness

This project analyzes phishing email examples and identifies common phishing indicators.

## Tools Used
- Google Docs
- Web Browser
- GitHub

## Analysis Approach
Three phishing email samples were analyzed by examining:
- Sender email address
- Suspicious URLs
- Urgent language
- Requests for sensitive information

## Repository Contents
- Phishing Detection Report
- Sample Email Evidence
- Documentation

## Conclusion
Phishing attacks rely on social engineering tactics. Awareness and careful inspection of emails can prevent most phishing attempts.
